# Quickcoords
QuickCoords adds a simple and fast way to save and jump to coordinates on PixelPlanet.

# Main features:
1. Save your current coordinates with a custom name

2. Instantly jump to any saved location

3. Copy direct coordinate links

4. Delete saved points

5. Movable and resizable panel

6. Compact collapse mode (turns into a small emoji button)

7. Remembers size, position, and state
